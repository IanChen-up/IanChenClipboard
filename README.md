# MacClipboard (Mac 本地剪贴板管理工具)

MacClipboard 是一款专为 macOS 设计的本地剪贴板管理工具。它通过原生 Mac 状态栏与现代化的高颜值 UI 相结合，为您提供高效、便捷的剪贴板历史记录和管理体验。

## ✨ 核心特性

- **原生状态栏集成**：隐藏 Dock 图标，安静驻留在屏幕右上角状态栏，一键点击即可召唤。
- **丰富的剪贴历史**：支持最多保存 150 条历史记录，支持文本和图片格式（PNG/TIFF）。
- **即时粘贴**：点击任意历史记录，自动将内容写入系统剪贴板，并带有友好的悬浮 Toast 成功提示。
- **现代化 UI**：采用 PyQt6 打造的无边框、圆角悬浮窗，告别死板的二级菜单。
- **明暗主题切换**：一键无缝切换 Light (明亮) 和 Dark (黑暗) 模式，自动适配各种元素颜色。
- **标签与常驻**：
  - **打标签**：支持为重要内容添加自定义标签（如 API、模板等）。
  - **常驻置顶**：一键置顶常用内容，即使产生新的复制记录，置顶内容依然在最前。
- **内嵌实时搜索**：无需弹出新窗口，直接在主界面搜索文本内容或自定义标签。
- **隐私安全**：纯本地应用，所有数据均存储在本地目录，不连接任何外部网络，安全可靠。

## 🚀 如何安装与运行

### 方式一：源码运行

1. 克隆或下载本仓库到本地。
2. 进入项目目录：
   ```bash
   cd mac-clipboard
   ```
3. 运行启动脚本（脚本会自动创建虚拟环境并安装依赖）：
   ```bash
   ./start.sh
   ```
4. 运行成功后，屏幕右上角状态栏会出现 📋 图标。

### 方式二：打包为 Mac App (独立运行)

您可以将本工具打包为 `.app` 文件，方便像普通 Mac 软件一样放入“应用程序”文件夹并设置开机自启。

1. 确保已安装所需的依赖：
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. 运行 `py2app` 打包命令：
   ```bash
   python setup.py py2app
   ```
3. 打包完成后，进入 `dist` 目录，您会看到 `MacClipboard.app`。
4. 将 `MacClipboard.app` 拖入您的 `Applications` (应用程序) 文件夹即可双击运行。

## 🛠 技术栈

- **Python 3**
- **PyQt6**：构建现代化且流畅的悬浮窗口界面。
- **AppKit / pyobjc**：调用 Mac 底层 API（`NSPasteboard`、`NSStatusBar` 等），实现 100% 稳定的原生剪贴板监控和图片处理，突破第三方库的限制。

## 💡 注意事项

- 首次运行打包后的 App 时，如果 macOS 提示安全性拦截，请前往“系统设置 -> 隐私与安全性”中点击“仍要打开”。
- 图片和文字的本地缓存数据保存在程序运行目录的 `data` 文件夹下。
